# Веб-интерфейс LexiconAI

Веб-интерфейс проекта LexiconAI представляет собой современное одностраничное приложение (SPA-like), обеспечивающее удобное взаимодействие с лингвистическим сервисом через браузер.

## Особенности интерфейса

1. **Интерактивный чат:** Ввод слов и получение ответов организованы в виде диалога, что делает процесс анализа привычным и интуитивно понятным.
2. **Реактивность без перезагрузок:** Благодаря использованию **HTMX**, поисковые запросы и обновление результатов происходят динамически. Страница никогда не перезагружается полностью, что обеспечивает мгновенный отклик.
3. **Визуальная индикация:**
    * **Цветовое кодирование:** Синонимы выделяются зеленым цветом, антонимы — красным, что позволяет быстро считывать информацию.
    * **Статусы процесса:** Пользователь видит индикатор "Анализирую...", пока ИИ формирует ответ.
4. **Адаптивность:** Интерфейс корректно отображается как на мониторах, так и на мобильных устройствах.
5. **Валидация:** Строгая проверка ввода на кириллицу и отсутствие пробелов.

## Как пользоваться

1. **Главная страница:** Перейдите по адресу [http://localhost](http://localhost) (или на домен вашего сервера).
2. **Поле ввода:** В центре экрана расположено поле для ввода слова.
3. **Запрос:** Введите слово (например, `Креативность`) и нажмите клавишу Enter или кнопку "Анализ".
4. **Результат:** Список ассоциаций появится в ленте чата ниже. Каждая карточка результата содержит 5 синонимов и 5 антонимов.

## Технологии фронтенда

* **FastAPI Templates:** Рендеринг на стороне сервера с использованием `Jinja2`.
* **HTMX:** AJAX-запросы и частичное обновление DOM без использования тяжелых JavaScript-фреймворков.
* **Tailwind CSS:** Современная стилизация и темная тема ("Dark Mode") по умолчанию.
* **Vanilla JS:** Минимальные скрипты для улучшения пользовательского опыта.

## Документация API

Для разработчиков доступен интерактивный интерфейс управления и тестирования API:

* **Swagger UI:** [http://localhost/docs](http://localhost/docs) — наглядный интерфейс для выполнения запросов.
* **ReDoc:** [http://localhost/redoc](http://localhost/redoc) — структурированная документация технических спецификаций.
