# Руководство по Telegram-боту LexiconAI

Telegram-бот является мобильным интерфейсом системы LexiconAI, предоставляющим мгновенный доступ к лингвистическому анализу слов прямо из мессенджера.

## Основные возможности

1. **Мгновенный анализ:** Отправьте любое слово боту, и он автоматически запустит процесс поиска синонимов и антонимов.
2. **Общая база знаний:** Бот подключен к той же базе данных, что и веб-интерфейс. Если слово уже было проанализировано кем-то другим, вы получите ответ мгновенно.
3. **Форматированный вывод:** Результаты представлены в удобном для чтения виде с использованием Markdown-разметки и эмодзи для разделения типов связей.

## Как пользоваться

### Начало работы

Найдите бота в Telegram и отправьте команду `/start`. Бот поприветствует вас и предложит ввести слово для анализа.

### Команды

- `/start` — Инициализация бота и краткое приветствие.
- `/help` — Получение краткой инструкции по использованию.

### Процесс анализа

Просто введите одно слово (например, `Прогресс`) и отправьте его сообщением.

1. Бот пришлет статус: "Анализирую слово **Прогресс**...".
2. Система обратится к ИИ (или кэшу), сформирует список из 10 ассоциаций.
3. Бот обновит сообщение, предоставив структурированный список.

### Пример ответа
>
> Результат для: **Прогресс**
>
> ✅ **Синонимы**:
> развитие, движение, рост, успех, подъем
>
> ❌ **Антонимы**:
> регресс, упадок, стагнация, застой, деградация

## Технические особенности

- **Фреймворк:** Бот построен на базе `aiogram 3.3.0` (асинхронный подход).
- **Обработка ошибок:** Система корректно обрабатывает пустые запросы, слишком длинные слова (более 50 символов) и технические сбои на стороне LLM-провайдеров.
- **Логирование:** Все действия бота фиксируются в централизованной системе логирования проекта для обеспечения стабильности и анализа популярности слов.

## Настройка и запуск (для администраторов)

Для работы бота необходимо:

1. Получить `TELEGRAM_BOT_TOKEN` у [@BotFather](https://t.me/BotFather).
2. Указать токен в файле `.env` в корне проекта:

    ```env
    TELEGRAM_BOT_TOKEN=your_bot_token_here
    ```

3. Убедиться, что сервис `bot` запущен в Docker-контейнере или через `main.py` в папке бота:

    ```bash
    python -m src.bot.main
    ```
